# Projeto Dundie Rewards

[![CI](https://github.com/GeovaneParedes/dundie-rewards/actions/workflows/main.yml/badge.svg)](https://github.com/GeovaneParedes/dundie-rewards/actions/workflows/main.yml)

Nós fomos contratados pela Dunder Mifflin, grande fabricante de papéis para
desenvolver um sistema de recompensas para seus colaboradores.

Michael, o gerente da empresa quer aumentar a motivação dos funcionários oferendo
um sistema de pontos que os funcionários podem acumular de acordo com as suas metas
atingidas, bonus oferecidos pelo gerente e os funcionários podem trocar pontos 
entre si.

O funcionário pode uma vez por ano, resgatar seus pontos em um cartão de credito
para gastar onde quiserem.

Acordamos em contrato que o MVP (Mininum Viable Product) será uma versão para ser 
executada no terminal e que no futuro terá as interfaces UI, Web e API.

Os dados dos funcionários atuais serão fornecidos em um arquivo, que pode ser
no formato .csv, txt, ou .json e este mesmo arquivo poderá ser usado para futuras
versões. "Nome, Depto, Cargo, Email"
